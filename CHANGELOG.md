# Changelog

All notable changes to this skill repository are recorded here.

## 2026-04-15

- Added script-assisted review support to `sci-review`.
- Added `check_refs.py` for label and hard-coded reference checks.
- Added `check_wording.py` for weak or subjective academic wording checks.
- Added `check_tex_structure.py` for template leftovers and simple structure checks.
- Added `check_citations.py` for citation consistency checks across `.bib` files and inline `thebibliography` / `\bibitem` entries.
- Updated review rules to prefer Markdown output in Chinese while keeping English SCI review standards.
- Added rules for equation punctuation, label-based references, and stricter wording review.
- Documented Codex and OpenCode installation steps.

## 2026-03-30

- Initialized the repository and started tracking `sci-review`.
